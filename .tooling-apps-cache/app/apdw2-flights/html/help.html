Flights Search

Submit a request to http://localhost:8081/flights?code=<>&airline=<>&simulate=<>

Default Values are:

  • code: SFO
  • airline: american
  • mock: true

Allowed airport codes are CLE, LAX, PDX, and SFO. All other values default to SFO.

If simulate = true, then all three airline services are simulated locally within the Mule application. This allows the Flights search to work when the computer is offline. If mock = false, then external requests are made to the three MuleSoft training web services for simulated American, Delta, and United airlines flight information.

Here are some common links used in the course:

  • http://locahost:8081/flights?code=SFO&airline=american
  • http://locahost:8081/flights?code=LAX&airline=delta
  • http://locahost:8081/flights?code=CLE&airline=united
  • http://locahost:8081/flights?code=LAX&airline=all
  • http://locahost:8081/flights?code=CLE&airline=united

Use real MuleSoft Training hosted web services

  • http://locahost:8081/flights?code=SFO&airline=american&mock=false
  • http://locahost:8081/flights?code=LAX&airline=delta&mock=false
  • http://locahost:8081/flights?code=CLE&airline=united&mock=false
  • http://locahost:8081/flights?code=LAX&airline=all&mock=false
  • http://locahost:8081/flights?code=CLE&airline=united
